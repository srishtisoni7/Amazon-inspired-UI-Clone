# Amazon Inspired UI Clone

## Description

This project is a simple Amazon-inspired UI clone created for learning and practice.
It focuses on building the basic homepage layout and product list sections using frontend technologies.

The project currently includes the basic structure and design of the homepage along with some product list items.

## Features

* Amazon-style homepage layout
* Navigation bar UI
* Product list sections
* Basic responsive layout
* Clean and simple UI design

## Technologies Used

* HTML
* CSS
* JavaScript (basic)

## Project Status

This project currently focuses only on the **UI design**.
Functional features like **Add to Cart, Login, and Checkout are not implemented yet**.

## Future Improvements

If I get more time, I plan to improve this project by adding:

* Add to Cart functionality
* Product details page
* Better responsiveness
* More UI sections
* JavaScript functionality

## How to Run

1. Clone or download this repository
2. Open the project folder
3. Open `index.html` in your browser

## Purpose

This project was built for practice to improve my frontend development and UI design skills.
